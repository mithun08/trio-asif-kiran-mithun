The Problem

A staffing manager matches consultants to open roles by hand in a spreadsheet — slow and inconsistent.

We build a ranked, explainable shortlist
so humans decide faster, with better information.

Design Philosophy

Four principles we never break

01
Deterministic core, LLM at the edges
Filtering, weighting, and ranking are pure Python arithmetic. LLM is used only where unstructured text demands it.
02
The LLM never sets a rank
LLM extracts signals from prose. The rank comes from a formula in scoring/ranker.py — always traceable to config.
03
PII scrubbed before any external call
Presidio + spaCy run locally first. A post-scrub regex gate is fail-closed. No identifiable data leaves the machine unredacted.
04
Every stage is independently testable
Each pipeline stage takes typed Pydantic inputs and returns typed outputs. No hidden state, no shared globals.
