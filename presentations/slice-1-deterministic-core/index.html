The six dimensions5 / 9

Tuned to the staffer, not hard-codedThe weights are configuration, not code

Every weight lives in a config file and must sum to 100%. A staffing manager can re-prioritise for the situation — no code change, no redeploy — and the shortlist re-orders to match.

  • Hard-to-find skill? Lean on skill match.
  • Must fill this week? Lean on availability + supply.

An invalid mix (one that doesn't sum to 100%) is rejected with a clear message — the engine can't be silently mis-tuned.

Bands & signals6 / 8

The subtle part"2 of 6 strong" can rank below "1 of 6 strong"

The signal count is a coarse summary. The actual rank is the weighted score — which respects magnitude and weight.

Karan's two Strong signals are availability and supply state — the lowest weight at 5%. Vikram's edge is on skill — the heaviest at 35%. One heavily-weighted near-miss beats two lightly-weighted Strongs.
